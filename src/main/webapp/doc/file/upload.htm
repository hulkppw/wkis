file/upload

上传文件接口

URL

http://xxx/file/upload

支持格式

JSON

HTTP请求方式

GET

是否需要登录

否

请求参数

  必选 类型及范围 说明
key true string 加密key
source true int32 文件来源 0：web网页，1：安卓TV客户端，2：安卓手机平板客户端，3：苹果客户端，100：其他

注意事项

  • 上传文件通过http　post方式提交，放在http消息body中传输

返回结果

JSON示例

{
    "result":"ok",
    "id":"89993E5C19620BB2B97784C7AA879AF6",
    "kus_id":"cn"
}

关于错误返回值与错误代码，参见 错误代码说明

返回字段说明

返回值字段 字段类型 字段说明
result string 结果, ok:成功,failure:失败
id string 文件id，用于获取文件信息和删除使用
kus_id string 当前kus编号

其他

无
