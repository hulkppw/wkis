imusic/v2/first_channel

获取第一层频道列表及相关信息

URL

http://xxx/imusic/v2/first_channel

支持格式

JSON

HTTP请求方式

GET

是否需要登录

否

请求参数

  必选 类型及范围 说明
language true int32 语言 0：简体中文，1：繁体中文

注意事项

  • 无
