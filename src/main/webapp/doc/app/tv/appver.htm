app/tv/appver

获取TV APP版本信息

URL

http://xxx/app/tv/appver

支持格式

JSON

HTTP请求方式

GET

是否需要登录

否

请求参数

  必选 类型及范围 说明
packagename true string App包名

注意事项

  • 无
