app/tv/apps

通过栏目id获取TV APP列表

URL

http://xxx/app/tv/apps

支持格式

JSON

HTTP请求方式

GET

是否需要登录

否

请求参数

  必选 类型及范围 说明
columnid true int32 栏目编号 0：全部（用于搜索），1:推荐，2：排行，11：新上架，12：装机必备，17：首页底部推荐列表，21：影音排行，22：应用排行，23： * 游戏排行，31：影音总分类，32：应用总分类，33：游戏总分类，其他栏目ID从栏目的推荐和分类列表获取
pagesize true int32 每页数据条数
pageindex true int32 请求的页码(从1开始)
order true int32 0：上架时间，1：下载量
spell true string app首字拼音

注意事项

  • 无
