pk/arena

pk/arena

热门擂台

URL

http://xxx/pk/arena

支持格式

JSON

HTTP请求方式

GET

是否需要登录

否

请求参数

  必选 类型及范围 说明
language true int32 语言 0：简体中文，1：繁体中文

注意事项

  • 如果当前擂台没有打擂歌曲（歌曲没有添加、或者歌曲的生效时间没有在今天），返回的字段song_id,song_name,singer_name，都为空字符串。客户端应该处理这种情况，提示用户没有擂台歌曲，请选择其他擂台

返回字段说明

返回值字段 字段类型 字段说明
id int32 擂台id
name string 擂台名称
image string 擂台图片地址
arena_song_id int32 当前擂台歌曲id
song_id string 歌曲曲库编号，用于点歌
song_name string 歌曲名称
singer_name string 歌手名称
