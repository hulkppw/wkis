pk/star

获取pk之星

URL

http://xxx/pk/star

支持格式

JSON

HTTP请求方式

GET

是否需要登录

否

请求参数

  必选 类型及范围 说明
language true int32 语言 0：简体中文，1：繁体中文

注意事项

  • 无

返回结果

[
    {
        "nickname":"爸爸",
        "singer_song_id":1,
        "index":1,
        "songname":"喜欢你",
        "head":"http://ww1.sinaimg.cn/mw600/9c824e06gw1em1dz0lrk1j20bn0hsdhh.jpg",
        "listen_count":100,
        "like_count":1000
    },
    {
        "nickname":"爸爸",
        "singer_song_id":1,
        "index":2,

    ...

关于错误返回值与错误代码，参见 错误代码说明

返回字段说明

返回值字段 字段类型 字段说明
nickname string 歌手昵称
singer_song_id int 歌手歌曲id
index int 歌手区块编号（决定图片位置）
songname string 歌曲名称
head string 歌手头像地址
listen_count int 试听次数
like_count int 点赞次数
