singerlist

获取歌手

URL

http://xxx/singerlist

支持格式

JSON

HTTP请求方式

GET

是否需要登录

否

请求参数

  必选 类型及范围 说明
language true int32 返回数据的语言类型（0：简体中文，1：繁体中文）
pageindex true int32 请求页码（从1开始）
pagesize true int32 每页数据条数
kind true int32 歌手分类0：全部查询 1：男歌星 2：女歌星 3：乐队组合 4：男/女 5：女/女 6：男/男
stroke true string 格式例如：“1”、“1-”或“1-2”， 歌手首字笔划，指定范围查找
spell true string 歌手注音,空值时查询全部记录
name true string 歌手名称
words true string 格式例如：“1”、“1-”或“1-2”，歌名字数，指定范围查找
firstword true string 歌手首字，可以是单字，也可以是多字

注意事项

  • 无

关于错误返回值与错误代码，参见 错误代码说明

返回值字段 字段类型 字段说明

recordcount int32 总记录数
id string 歌手编号
name string 歌手名称
kind int32 歌手分类 1：男歌星 2：女歌星 3：乐队组合 4：男/女 5：女/女 6：男/男

其他
