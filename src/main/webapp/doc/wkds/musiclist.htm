musiclist

musiclist

获取歌单

URL

http://xxx/musiclist

支持格式

JSON

HTTP请求方式

GET

是否需要登录

否

请求参数

  必选 类型及范围 说明
language true int32 返回数据的语言类型（0：简体中文，1：繁体中文）
pageindex true int32 请求页码（从1开始）
pagesize true int32 每页数据条数
type true int32 歌曲分类 0：全部查询 ,1:金曲排行, * 2:体验试唱,3:情歌,4:民歌,5:怀旧,6:舞曲,7:Rap,8:校园,9:本月上架,10:本月劲榜,11:最新上传 * ,12 * :儿歌,13:可录增值,14:体验曲库,15:红歌,16:老歌,17:流行,18:人生导唱,19:欢乐节庆,20:周杰伦 * ,21:邓丽君, * 22:刘德华,23:情歌KODII,24:经典老歌KODII,25:流行歌曲KODII,26:中国好声音（第一季 * ）,27:中国好声音（第二季）,28:国庆歌曲,29:中秋节歌曲,30:深情对唱,31:发烧话题
stroke true string 格式例如：“1”、“1-”或“1-2”， 歌曲首字笔划，指定范围查找
address true int32 歌曲地区 1台灣, 2香港, 3內地, 4亞州,馬來西亞, 新加坡, 日本,韩国,泰国 5歐美, 0:全部
singerid true string 歌手编号 为空查全部
words true string 格式例如：“1”、“1-”或“1-2”，歌曲字数，指定范围查找
spell true string 歌名拼音 为空查全部
languagecode true int32 语言类别 1:國語,2:台語,3:粵語,4:韓語,5:日語,6:英語,7:其他語種,8:客语,9:儿语 ,0:全部
order true int32 排序 0：更新，1：热度
name true string 歌名搜索 为空查全部
isis9 true int32 搜索是否有人声的歌曲 0：没有人声 1：有人声 2：全部
songid true string 歌单编号
firstword true string 歌曲首字，可以是单字，也可以是多字

返回字段说明

返回值字段 字段类型 字段说明
pagecount int32 总页数
recordcount int32 总记录数
language int32 歌曲所属语言 1:國語,2:台語,3:粵語,4:韓語,5:日語,6:英語,7:其他語種,8:客语,9:儿语 ,0:全部
singerid string 歌手编号
singer string 歌手名称
singertype int32 歌手类型（1男歌手，2女歌手，3乐团组合，4男女合唱，5女女合唱，6男男合唱，7其它）
id string 歌曲编号
name string 歌曲名称
isis9 int32 人声导唱 1：有,0:无
spell string 歌曲拼音
s1 int32 获取主情景
s2 int32 获取二级情景
