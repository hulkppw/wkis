multiscreen/favorite

获取我的收藏

URL

http://xxx/multiscreen/favorite

支持格式

JSON

HTTP请求方式

GET

是否需要登录

是

请求参数

  必选 类型及范围 说明
session true string session
pageindex true int32 请求页码（从1开始）
pagesize true int32 每页数据条数
language true int 语言０：简体中文，１：繁体中文

注意事项

  • 无

返回字段说明

返回值字段 字段类型 字段说明
id int32 收藏歌曲编号，用于删除
pagecount int32 总页数
recordcount int32 总记录数
languagecode int32 歌曲所属语言 1:國語,2:台語,3:粵語,4:韓語,5:日語,6:英語,7:其他語種,8:客语,9:儿语 ,0:全部
singerid string 歌手编号
singer string 歌手名称
singertype int32 歌手类型（1男歌手，2女歌手，3乐团组合，4男女合唱，5女女合唱，6男男合唱，7其它）
songid string 歌曲编号
name string 歌曲名称
isis9 int32 人声导唱 1：有,0:无
spell string 歌曲拼音
