player/theme/music/list

player/theme/music/list

获取主题歌单

URL

http://xxx/player/theme/music/list

支持格式

JSON

HTTP请求方式

GET

是否需要登录

否

请求参数

  必选 类型及范围 说明
themeid true int32 主题编号
pageindex true int32 请求页码（从1开始）
pagesize true int32 每页数据条数

注意事项

  • 无

返回字段说明

返回值字段 字段类型 字段说明
pagecount int32 总页数
recordcount int32 总记录数
language int32 歌曲所属语言 1:國語,2:台語,3:粵語,4:韓語,5:日語,6:英語,7:其他語種,8:客语,9:儿语 ,0:全部
singerid string 歌手编号
singer string 歌手名称
singertype int32 歌手类型（1男歌手，2女歌手，3乐团组合，4男女合唱，5女女合唱，6男男合唱，7其它）
id string 歌曲编号
name string 歌曲名称
isis9 int32 人声导唱 1：有,0:无
spell string 歌曲拼音
scene1 int32 获取主情景
scene2 int32 获取二级情景
