player/history/list

获取点唱历史记录列表

URL

http://xxx/player/history/list

支持格式

JSON

HTTP请求方式

GET

是否需要登录

是

请求参数

  必选 类型及范围 说明
pagesize true int32 每页几条
pageindex true int32 第几页
session true String session（用来获取userid）

注意事项

  • 无
