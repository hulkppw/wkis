player/tone/list

获取我的定调列表

URL

http://xxx/player/tone/list

支持格式

JSON

HTTP请求方式

GET

是否需要登录

是

请求参数

  必选 类型及范围 说明
pagesize true int32 每页几条
pageindex true int32 第几页
session true String session（用来获取userid）

注意事项

  • 无

返回结果

JSON示例

{"pagecount":3,
    "recordcount":6,
    "list":[
            {"id":190,"userid":"1000000","singerid":4,"topnote":3,"highkey":63,"ddtype":0,"sex":1,"ismusictraining":0,"autokeytip":0,"touxiang":"","result":"","hkfreqz":"311","hasdd":"0","createtime":1370409009000},
            {"id":189,"userid":"1000000","singerid":5,"topnote":4,"highkey":67,"ddtype":0,"sex":1,"ismusictraining":0,"autokeytip":0,"touxiang":"","result":"","hkfreqz":"392","hasdd":"0","createtime":1370400082000}
     ]
}

关于错误返回值与错误代码，参见 错误代码说明

返回字段说明

返回值字段 字段类型 字段说明
pagecount int32 总页数
recordcount string 总数
id int 编号
userid string 本人用户ID
singerid int 定调歌手ID
topnote int 歌曲最高音
highkey int 人声最高音
ddtype int 1:最高音定调；2:歌曲定调；
sex int 性别1：男声；2：女声；
ismusictraining int 是否受过声乐训练
autokeytip int 0：没有升降调系统提示；1：选择了手动升降调提示；2：选择了自动升降调
touxiang string 定调对应的头像名字
result string 定调结果
hkfreqz string 频率数值
hasdd string 是否已经定过调
createtime date 时间

其他

无
