player/tone/add

添加我的定调

URL

http://xxx/player/tone/add

支持格式

JSON

HTTP请求方式

GET

是否需要登录

是

请求参数

  必选 类型及范围 说明
singerid true int32 定调歌手ID
topnote true int32 歌曲最高音
highkey true int32 人声最高音
ddtype true int 1:最高音定调；2:歌曲定调
sex true int 性别1：男声；2：女声；
ismusictraining true int 是否受过声乐训练
autokeytip true int 0：没有升降调系统提示；1：选择了手动升降调提示；2：选择了自动升降调
touxiang true String 定调对应的头像名字
result true String 定调结果
hkfreqz true String 频率数值
hasdd true String 是否已经定过调
session true String session（用来获取userid）

注意事项

  • 无

返回结果

JSON示例

{result:ok}

关于错误返回值与错误代码，参见 错误代码说明

返回字段说明

返回值字段 字段类型 字段说明
result int32 1.ok是成功 2.error是失败 3.参数不正确

其他

无
