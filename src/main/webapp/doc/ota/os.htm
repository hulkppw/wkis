ota/os

ota/os

获取操作系统更新信息

URL

http://xxx/ota/os

支持格式

JSON

HTTP请求方式

GET

是否需要登录

否

请求参数

  必选 类型及范围 说明
devicecode true string 设备类型编号
utc true int64 当前系统utc时间

注意事项

  • 无

返回结果

JSON示例

 {
    "name": "OTT-2.0.19",
    "version": "20140522151241",
    "md5": "076E3CAED758A1C18C91A0E9CAE3368F",
    "size": 879394,
    "url": "http://xxx.xxx.com/resource/ota/upload/20140522/4ab55998f10442e5952b1b2ef09a1997.bin",
    "code": "123456898",
    "description": "",
    "utc":124567
}

关于错误返回值与错误代码，参见 错误代码说明

返回字段说明

返回值字段 字段类型 字段说明
name string 系统名称
version string 版本号
md5 string 系统文件md5
size int 系统文件大小
url string 下载地址
code string 当前系统唯一编号（用于下载系统资源使用）
description string 系统说明
utc int64 系统utc时间

其他

无
