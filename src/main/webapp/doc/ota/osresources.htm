ota/osresources

ota/osresources

获取操作系统资源列表

URL

http://xxx/ota/osresources

支持格式

JSON

HTTP请求方式

GET

是否需要登录

否

请求参数

  必选 类型及范围 说明
oscode true string 操作系统编号

注意事项

  • 无

返回字段说明

返回值字段 字段类型 字段说明
id int32 资源编号（系统中唯一）
version string 版本号
md5 string 系统文件md5
path string 下载到本地的路径
size int 系统文件大小
filename string 系统文件名称
url string 下载地址
description string 系统说明
