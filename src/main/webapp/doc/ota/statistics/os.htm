ota/statistics/os

操作系统更新统计

URL

http://xxx/ota/statistics/os

支持格式

JSON

HTTP请求方式

GET

是否需要登录

否

请求参数

  必选 类型及范围 说明
version true string 当前操作系统版本
next_version true string 新版操作系统版本
wifimac false string wifi MAC地址（去掉冒号转小写）
mac true string 有线MAC地址（去掉冒号转小写）
sn false string 产品编号
koiccode false string 爱唱声卡编号
firmware false string 韧体版本号

注意事项

  • 无

返回结果

JSON示例

 {
    "result": 0,
    "message": "成功"
}

关于错误返回值与错误代码，参见 错误代码说明

返回字段说明

返回值字段 字段类型 字段说明
result int32 结果编号（如果不为0请查看 错误代码说明）
message string 返回说明

其他

用户客户端下载操作系统文件成功后调用
